## 🚀 Overview

**O.S.T Manager** is a next-generation desktop application designed to provide seamless management, unlocking, and synchronization for your AAA Steam game library. It automates tedious tasks such as Manifest Injection and Denuvo Ticket Extraction, allowing you to focus on what matters most: playing your games.

## 🌟 Key Features

* **🎮 Automated Store & Game Injector**
  Browse a rich catalog of over 50+ AAA titles. With a single click, O.S.T Manager handles Lua injection and background manifest downloading seamlessly. No manual patching or file replacement required.

* **☁️ CloudRedirect Sync**
  Bypass Steam's default cloud storage limitations. Automatically back up and synchronize your local save games directly to your Google Drive or OneDrive. Never lose your progress again.

* **🎫 Denuvo Ticket Extractor (VIP Feature)**
  Easily dump AppTickets and ETickets from your owned games with a single click. Share them safely with the community without exposing your personal account details.

* **🌍 Multi-language Support**
  Fully localized interface supporting both English and Vietnamese (Tiếng Việt) out of the box.

* **🎨 Cyberpunk Aesthetic UI**
  Built with a breathtaking Hacker/Matrix-inspired UI, featuring glassmorphism, dynamic neon glitches, and smooth micro-animations.

## 📥 Installation

1. Navigate to the **[Releases](../../releases/latest)** tab on the right.
2. Download the latest `O.S.T_Manager_Setup.exe` file.
3. Run the installer and follow the on-screen instructions.
4. Launch the app and log in!

> **Note:** Windows Defender might flag the installer due to the code-injection techniques used for Steam automation. This is a false positive. Please add it to your exclusions list.
